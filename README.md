# GorillaTagMLModExample

MelonLoader Mod Example for Gorilla Tag, includes Setup, Harmony Patching, Asset Bundles, and Modded Room support. 

> [!WARNING]
> Modded room support for the template will be added as soon as it is ported to MelonLoader.
> 
> Gorilla Tag also **REQUIRES** MelonLoader 0.7.2.
>
> This should really only be used as an example, not a template. A new template will be created soon that has minimal things, just for it to function.
