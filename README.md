# GorillaTagMLTemplate

MelonLoader Mod Template for Gorilla Tag, includes Setup, Harmony Patching, Asset Bundles, and Modded Room support. 

> [!WARNING]
> Modded room support for the template will be added as soon as it is ported to MelonLoader.
> 
> Gorilla Tag also **REQUIRES** MelonLoader 0.7.2 Nightly.
> You can get this version by ticking "Enable nightly builds" in the MelonLoader installer.
